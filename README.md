# PAD - Pine At Dawn Team Portfolio

> Team portfolio website for PAD (Pine At Dawn)

## Overview

PAD is a team portfolio website featuring team member profiles, project showcases, an AI English tutor, and a guestbook.

## Key Features

- **Team Portfolio**: Member profiles and project showcase
- **AI English Tutor**: OpenAI-powered conversation practice
- **Guestbook**: Visitor notes feature
- **Multilingual**: Korean/English toggle

## Tech Stack

| Category | Technologies |
|----------|-------------|
| Frontend | Next.js 15, React, TypeScript, TailwindCSS |
| Backend | Next.js API Routes, MongoDB |
| AI | OpenAI GPT API |
| Deployment | Vercel |

## Quick Start

```bash
cd frontend
npm install
npm run dev
```
